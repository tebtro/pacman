# Pac-Man

### Platforms:
- Windows (x86, x64), using Win32-API
- Linux (x86, x64) , using SDL2

### Features:
- Platform layer
- Controller support (XInput)
- Load and render bitmaps
- Render bitmap font
- Titlescreen
